# Riichi Mahjong Guide

A Guide for Learning and Playing Japanese Mahjong

## About

This repository contains PDFs, images, and resources for learning Japanese Riichi Mahjong. The guide is designed for **beginners to intermediate players** and includes step-by-step learning materials.

## Contents

- 0 Overview & Guide Introduction.pdf  
- 1 Mahjong Tiles & Winning Hands.pdf  
- 2 Setup & Gameplay Reference.pdf  
- 3 Introduction to Tile Calls.pdf  
- 4 Essential Terms & Rules.pdf  
- 5 List of Yaku.pdf  
- 6 How to Calculate a Hand's Score.pdf  
- 7 Additional Rules & Supplementary Material.pdf
